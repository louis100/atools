## Project Setup

```sh
npm install
```

### run in browser

```sh
npm run dev
```
### run in Electron
```sh
npm run start:e
```

### Package into an exe
```sh
npm run make:e
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
